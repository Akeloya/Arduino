# Arduino
Простой эмулятор ардуино для разработки скетчей для платы и наглядной визуализации работы схемы.
## Цель
- Реализовать программную среду для эмуляции работы реальной схемы на ардуино с цельу отладки скетчей без порчи платы и обучения в отсутствии физических модулей.
### Возможности
- Интерпретация скетча, проверка ошибок и выполнение кода
- Эмуляция работы платы и связанных компонент
- Добавление компонент с возможностью описания поведения
- Схематическая визуализация работы схемы.
